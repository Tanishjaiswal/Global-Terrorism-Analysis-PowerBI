# Global Terrorism Analysis (2007–2020)

## 📊 Project Overview
This project analyzes global terrorism data using Microsoft Power BI. The objective is to identify patterns, trends, and insights related to terrorist activities across different regions and time periods.

## 🎯 Objectives
- Analyze terrorism trends over time
- Identify high-risk regions and countries
- Study attack types and their impact
- Perform casualty analysis

## 🛠 Tools Used
- Microsoft Power BI
- Excel (Data Cleaning)

## 📁 Dataset
Global Terrorism Database (GTD)  
https://www.start.umd.edu/download-global-terrorism-database

## 📌 Key Insights
- Terrorism incidents are concentrated in specific regions
- Attack frequency and severity vary significantly
- Certain attack types are more lethal
